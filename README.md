# Ardena Raiche — Data Analytics & Visualization Portfolio

Live Site: **https://ardenaraiche-svg.github.io/**

A personal portfolio website showcasing data analytics, visualization, and business intelligence work.  
Built with a clean, responsive UI and designed for recruiter-friendly scanning.

---

## ✨ Highlights

- Responsive, mobile-friendly layout
- Sticky navigation for quick section access
- Dark/Light theme toggle with saved preference (`localStorage`)
- Accessibility-focused enhancements:
  - Skip link
  - Visible keyboard focus states
  - Semantic HTML landmarks
- SEO/social metadata:
  - Meta description
  - Open Graph tags
  - Twitter card tags
  - Canonical URL
- Structured data (`JSON-LD`) for discoverability

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3** (custom properties, responsive grid/flex)
- **Vanilla JavaScript** (theme toggle + persistence)
- **GitHub Pages** for hosting

## 🚀 Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/ardenaraiche-svg/ardenaraiche.github.io.git
   cd ardenaraiche.github.io
   ```
2. Open `index.html` in your browser.

> No build step required — this is a static site.

---

## 🔄 Deployment

This site is deployed with **GitHub Pages** from the `main` branch.

After pushing changes:
1. Go to **Settings → Pages** in the repo.
2. Ensure source is set to `main` branch (root).
3. Wait ~1–3 minutes for deployment.
